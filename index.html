About Alaire

Alaire is a powerful bot designed to help you manage your Discord server with ease. It includes fun features, moderation tools, and more.

[-----------]

Alaire is made with BotForge, a cilent made with Discord.js for nodejs. Sadly, BotForge is in beta which means, Alaire is in beta too.

Commands

  • /utilty help - Displays a list of commands
  • /moderation ban [reason] [user] - Bans a user from the server
  • /moderation kick [reason] [user] - Kicks a user in the server
  • /utilty embed-creator [title] [description] - Makes a custom embed
  • /music play [title] - Plays music from any playfrom eg. Youtube
  • /utilty info - Get information about the bot
